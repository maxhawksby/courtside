# courtside

Custom basketball coaching platform for a private client — a functional GameChanger-class app
(scheduling, scorekeeping, stats, messaging, live-game following) built around the thing
GameChanger doesn't have: an org-wide database of parents, players, and teams.

> **`courtside` is a working codename.** The public app name is chosen after a trademark
> knockout search (must contain nothing near "GameChanger"/"GC"). See `docs/CLIENT-KICKOFF.md`.

**Status**: early scaffold — Phase 0 (foundations & store unblocking).

- Plan of record: `docs/BRIEF.md` · architecture: `docs/ARCHITECTURE.md` · phases:
  `docs/ROADMAP.md` · legal: `docs/COMPLIANCE.md`
- Targets: iOS + Android (Expo/React Native) + web admin console. Backend: Supabase.
